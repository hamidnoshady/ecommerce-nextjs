# AGENTS.md

## Project intent
Build a production-ready headless WordPress + WooCommerce storefront using Next.js and a shadcn-inspired luxury/editorial design system.

## Rules
- Prefer maintainable architecture over shortcuts.
- Keep UI premium, minimal, and consistent.
- Do not introduce random visual styles outside the design system.
- Use reusable components and typed data mapping.
- Keep WordPress content logic separate from WooCommerce commerce logic.
- Treat mobile UX as first-class.
- Document assumptions whenever live credentials or store config are required.
- Favor small, reviewable commits.
